Welcome to a modern front-end

Adopting USWDS has never been easier.

Clone or use this template

When creating a new repo, you can select this template (listed as "NEFSC/ITD-DDB-USWDS"), or visit the ITD-DDB-USWDS repo and clone or download this repo.

Pages will continue to be added over time, along with support for varying frameworks like CodeIgniter.

Forked from the official NEFSC template

No need to worry about this repo matching the NEFSC template, it's forked from the official repo and is fully compliant. You'll find the README.md, License.txt, and secretscan workflow already included.

Fully accessible

This repo has been reviewed and is fully compliant with 508C accessibility guidelines.

Use any pages or components you need

There are several sections (listed on the left) that you can copy and paste into your project. Remember you can always visit the USWDS website to find more components and other templates.

Contribute with feedback or pull requests

Make pull requests or submit issues to this repo to make it better and help others save time.

You can also join the USWDS Community for even more help and feedback.
